<h1>Python Essentials (v.2.0) &ndash; About the course curriculum</h1>

<br>

<p><strong>PCAP: Programming Essentials in Python</strong> (short form: <i>Python Essentials</i>) is a two-course series that covers all the basics of programming in Python, as well as general computer programming concepts and techniques, and the object-oriented approach.</p>

<p>The Python Essentials course series is divided into two parts:</p>
<ul>
<li><b>Python Essentials 1 (PE1): BASICS</b>, consisting of four modules;</li>
<li><b>Python Essentials 2 (PE2): INTERMEDIATE</b>, consisting of four modules.</li>
</ul>

<p>Each student has access to <strong>hands-on practice materials</strong>, <strong>labs</strong>, <strong>quizzes</strong>, and <strong>tests</strong> to learn how to utilize the skills and knowledge gained on the course and interact with some <strong>real-life programming tasks and situations</strong>.</p>

<p>Students who complete the course will be able to accomplish coding tasks related to the basics of programming in the Python language, and to understand the fundamental notions and techniques used in object-oriented programming.</p>

<p>Furthermore, they will be ready to attempt the following qualifications:</p>

<ul>
<li><a href="https://pythoninstitute.org/certification/pcep-certification-entry-level/" target="_blank">PCEP &ndash; Certified Entry-Level Python Programmer</a> (after completing Python Essentials 1)</li>
<li><a href="https://pythoninstitute.org/certification/pcap-certification-associate/" target="_blank">PCAP &ndash; Python Certified Associate Programmer</a> (after completing Python Essentials 2)</li>
</ul>

<p>from the <a href="https://pythoninstitute.org" target="_blank">OpenEDG Python Institute</a>.</p>
