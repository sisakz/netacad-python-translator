<h1>Using variables</h1>

<p>You're allowed to use as many variable declarations as you need to achieve your goal, like this:</p>

<codep class="syntax-color">var = 1
account_balance = 1000.0
client_name = 'John Doe'
print(var, account_balance, client_name)
print(var)
</codep>

<br>

<p>You're <b>not allowed to use a variable which doesn't exist</b> (in other words, a variable that was not assigned a value).</p>

<p>This example will <strong>cause an error</strong>:</p>

<codep><mark style="background-color:#e6f2ff;">v</mark>ar = 1
print(<mark style="background-color:#e6f2ff;">V</mark>ar)

</codep>


<br>

<p>We've tried to use a variable named <codel>Var</codel>, which doesn't have any value (note: <codel>var</codel> and <codel>Var</codel> are different entities, and have nothing in common as far as Python's concerned).</p>
