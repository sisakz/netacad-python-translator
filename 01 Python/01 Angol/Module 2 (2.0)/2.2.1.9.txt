<h1>Coding strings</h1>


<p>Now, the next question is: how do you embed an apostrophe into a string placed between apostrophes?</p>

<p>You should already know the answer, or to be precise, two possible answers.</p>

<p>Try to print out a string containing the following message:</p>


<codep>I'm Monty Python.</codep>

<br>


<p>Do you know how to do it? Click <i>Check</i> below to see if you were right:</p>

<p>As you can see, the backslash is a very powerful tool ‒ it can escape not only quotes, but also apostrophes.</p>
