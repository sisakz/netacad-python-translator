<h1>How to spoil and fix your code</h1>

<p>Watch all the windows carefully.</p>

<p>A new window appears – it says that the interpreter has encountered an EOF (<i>end-of-file</i>) although (in its opinion) the code should contain some more text.</p>

<p>The editor window shows clearly where it happened.</p>

<img class="lightbox" src="/uploads/media/default/0001/01/112b321a4d7620c67b0e037f8861fd71a9cb09df.png" alt="Python 3.7.0 IDLE - Console message after running a program with removed parenthesis from the print() function" width="80%">

<br>
<br>

</div>


<div class="small-12 large-6 columns">

<br>

<p>Fix the code now. It should look like this:</p>

<codep class="syntax-color">print("Hisssssss...")
</codep>

<br>

<p>Run it to see if it "hisses" again.</p>

<p>Let's spoil the code one more time. Remove one letter from the word <font face="Courier New">print</font>. Run the code by pressing <i>F5</i>. What happens now? As you can see, Python is not able to recognize the instruction.</p>
