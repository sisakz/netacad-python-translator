<h1>Key takeaways</h1>

<br>


<p>1. Python supports the following logical operators:</p>

<ul>
<li><codel>and</codel> &rarr; if both operands are true, the condition is true, e.g., <codel>(True and True)</codel> is <codel>True</codel>,</li>
<li><codel>or</codel> &rarr; if any of the operands are true, the condition is true, e.g., <codel>(True or False)</codel> is <codel>True</codel>,</li>
<li><codel>not</codel> &rarr; returns false if the result is true, and returns true if the result is false, e.g., <codel>not True</codel> is <codel>False</codel>.</li>
</ul>

<p>2. You can use bitwise operators to manipulate single bits of data. The following sample data:</p>

<ul>
<li><codel>x = 15</codel>, which is <codel>0000 1111</codel> in binary,</li>
<li><codel>y = 16</codel>, which is <codel>0001 0000</codel> in binary.</li>
</ul>

<p>will be used to illustrate the meaning of bitwise operators in Python. Analyze the examples below:</p>

<ul>

<li><codel>&amp;</codel> does a <i>bitwise and</i>, e.g., <codel>x &amp; y = 0</codel>, which is <codel>0000 0000</codel> in binary,</li>
<li><codel>|</codel> does a <i>bitwise or</i>, e.g., <codel>x | y = 31</codel>, which is <codel>0001 1111</codel> in binary,</li>
<li><codel>&tilde; </codel> does a <i>bitwise not</i>, e.g., <codel>&tilde; x = 240</codel>*, which is <codel>1111 0000</codel> in binary,</li>
<li><codel>^</codel> does a <i>bitwise xor</i>, e.g., <codel>x ^ y = 31</codel>, which is <codel>0001 1111</codel> in binary,</li>
<li><codel>&gt;&gt;</codel> does a <i>bitwise right shift</i>, e.g., <codel>y &gt;&gt; 1 = 8</codel>, which is <codel>0000 1000</codel> in binary,</li>
<li><codel>&lt;&lt;</codel> does a <i>bitwise left shift</i>, e.g., <codel>y &lt;&lt; 3 = </codel>, which is <codel>1000 0000</codel> in binary,</li>
</ul>

<br>

<p>* <codel>-16</codel> (decimal from signed 2's complement) -- read more about the <a href=https://en.wikipedia.org/wiki/Two%27s_complement target="_blank">Two's complement</a> operation.</p>
