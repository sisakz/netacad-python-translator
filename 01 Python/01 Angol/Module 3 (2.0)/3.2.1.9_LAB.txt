<h2>Scenario</h2>

<p>The <codel>break</codel> statement is used to exit/terminate a loop.</p>

<p>Design a program that uses a <codel>while</codel> loop and continuously asks the user to enter a word unless the user enters <codel>"chupacabra"</codel> as the secret exit word, in which case the message <codel>"You've successfully left the loop."</codel> should be printed to the screen, and the loop should terminate.</p>

<p>Don't print any of the words entered by the user. Use the concept of conditional execution and the <codel>break</codel> statement.</p>

<sampleSolution>
while True:
    word = input("You're stuck in an infinite loop!\nEnter the secret word to leave the loop: ")
    if word == "chupacabra":
        break
print("You've successfully left the loop!")
