<h1>The <font face="courier new">in</font> and <font face="courier new">not in</font> operators</h1>


<p>Python offers two very powerful operators, able to <strong>look through the list in order to check whether a specific value is stored inside the list or not</strong>.</p>

<p>These operators are:</p>

<codep>elem <mark style="background-color:#e6f2ff;">in</mark> my_list
elem <mark style="background-color:#e6f2ff;">not in</mark> my_list

</codep>

<br>


<p>The first of them (<codel>in</codel>) checks if a given element (its left argument) is currently stored somewhere inside the list (the right argument) ‒ the operator returns <codel>True</codel> in this case.</p>

<p>The second (<codel>not in</codel>) checks if a given element (its left argument) is absent in a list ‒ the operator returns <codel>True</codel> in this case.</p>
