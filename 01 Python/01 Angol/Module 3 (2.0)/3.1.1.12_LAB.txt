<h2>Scenario</h2>

<p>As you surely know, due to some astronomical reasons, years may be <i>leap</i> or <i>common</i>. The former are 366 days long, while the latter are 365 days long.</p>

<p>Since the introduction of the Gregorian calendar (in 1582), the following rule is used to determine the kind of year:</p>

<ul>
<li>if the year number isn't divisible by four, it's a <i>common year</i>;</li>
<li>otherwise, if the year number isn't divisible by 100, it's a <i>leap year</i>;</li>
<li>otherwise, if the year number isn't divisible by 400, it's a <i>common year</i>;</li>
<li>otherwise, it's a <i>leap year</i>.</li>
</ul>


<p>Look at the code in the editor â€“ it only reads a year number, and needs to be completed with the instructions implementing the test we've just described.<p>

<p>The code should output one of two possible messages, which are <codel>Leap year</codel> or <codel>Common year</codel>, depending on the value entered.</p>

<p>It would be good to verify if the entered year falls into the Gregorian era, and output a warning otherwise: <codel>Not within the Gregorian calendar period</codel>. Tip: use the <codel>!=</codel> and <codel>%</codel> operators.</p>

<p>Test your code using the data we've provided.</p>

<sampleSolution>
year = int(input("Enter a year: "))

if year < 1582:
	print("Not within the Gregorian calendar period")
else:
	if year % 4 != 0:
		print("Common year")
	elif year % 100 != 0:
		print("Leap year")
	elif year % 400 != 0:
		print("Common year")
	else:
		print("Leap year")
