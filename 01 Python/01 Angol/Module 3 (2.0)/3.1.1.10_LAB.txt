<h2>Scenario</h2>
<p><a href="https://upload.wikimedia.org/wikipedia/commons/b/bd/Spathiphyllum_cochlearispathum_RTBG.jpg">Spathiphyllum</a>, more commonly known as a peace lily or white sail plant, is one of the most popular indoor houseplants that filters out harmful toxins from the air. Some of the toxins that it neutralizes include benzene, formaldehyde, and ammonia.</p>

<p>Imagine that your computer program loves these plants. Whenever it receives an input in the form of the word <codel>Spathiphyllum</codel>, it involuntarily shouts to the console the following string: <codel>"Spathiphyllum is the best plant ever!"</codel></p>

<br>

<p>Write a program that utilizes the concept of conditional execution, takes a string as input, and:</p>

<ul>
<li>prints the sentence <codel>"Yes - Spathiphyllum is the best plant ever!"</codel> to the screen if the inputted string is <codel>"Spathiphyllum"</codel> (upper-case)</li>
<li>prints <codel>"No, I want a big Spathiphyllum!"</codel> if the inputted string is <codel>"spathiphyllum"</codel> (lower-case)</li>
<li>prints <codel>"Spathiphyllum! Not [input]!"</codel> otherwise. Note: <codel>[input]</codel> is the string taken as input.</li>
</ul>

<br>

<p>Test your code using the data we've provided for you. And get yourself a Spathiphyllum, too!</p>

<br>

<sampleSolution>
name = input("Enter flower name: ")

if name == "Spathiphyllum":
    print("Yes - Spathiphyllum is the best plant ever!")
elif name == "spathiphyllum":
    print("No, I want a big Spathiphyllum!")
else:
    print("Spathiphyllum! Not", name + "!")
