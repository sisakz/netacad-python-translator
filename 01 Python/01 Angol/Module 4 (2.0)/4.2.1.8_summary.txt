<h1>Key takeaways</h1>



<p>1. You can pass information to functions by using parameters. Your functions can have as many parameters as you need.</p>

<p>An example of a one-parameter function:</p>

<codep class="syntax-color">def hi(name):
    print("Hi,", name)

hi("Greg")
</codep>
<br>

<p>An example of a two-parameter function:</p>

<codep class="syntax-color">def hi_all(name_1, name_2):
    print("Hi,", name_2)
    print("Hi,", name_1)

hi_all("Sebastian", "Konrad")
</codep>
<br>

<p>An example of a three-parameter function:</p>

<codep class="syntax-color">def address(street, city, postal_code):
    print("Your address is:", street, "St.,", city, postal_code)

s = input("Street: ")
p_c = input("Postal Code: ")
c = input("City: ")

address(s, c, p_c)
</codep>
<br>

<p>2. You can pass arguments to a function using the following techniques:</p>

<ul>
<li><strong>positional argument passing</strong> in which the order of arguments passed matters (Ex. 1)</li>
<li><strong>keyword (named) argument passing</strong> in which the order of arguments passed doesn't matter (Ex. 2)</li>
<li>a mix of positional and keyword argument passing (Ex. 3.)</li>

</ul>

<codep class="syntax-color">Ex. 1
def subtra(a, b):
    print(a - b)

subtra(5, 2)    # outputs: 3
subtra(2, 5)    # outputs: -3


Ex. 2
def subtra(a, b):
    print(a - b)

subtra(a=5, b=2)    # outputs: 3
subtra(b=2, a=5)    # outputs: 3

Ex. 3
def subtra(a, b):
    print(a - b)

subtra(5, b=2)    # outputs: 3
subtra(5, 2)    # outputs: 3
</codep>
<br>

<p>It's important to remember that <strong>positional arguments mustn't follow keyword arguments</strong>. That's why if you try to run the following snippet:</p>

<codep style="color:red;">def subtra(a, b):
    print(a - b)

subtra(5, b=2)    # outputs: 3
subtra(<mark style="background-color:#e6f2ff;">a=5, 2</mark>)    # Syntax Error</codep>

<br>
<p>Python will not let you do it by signalling a <codel>SyntaxError</codel>.</p>

<p>3. You can use the keyword argument-passing technique to <strong>pre-define</strong> a value for a given argument:</p>

<codep class="syntax-color">def name(first_name, last_name="Smith"):
    print(first_name, last_name)

name("Andy")    # outputs: Andy Smith
name("Betty", "Johnson")    # outputs: Betty Johnson (the keyword argument replaced by "Johnson")
</codep>
