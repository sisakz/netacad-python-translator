<h1>Some simple functions: factorials</h1>

<p>Another function we're about to write is <strong>factorials</strong>. Do you remember how a factorial is defined?</p>

<p style="font-family:courier new;">0! = 1 (yes! it's true)<br>
1! = 1<br>
2! = 1 * 2<br>
3! = 1 * 2 * 3<br>
4! = 1 * 2 * 3 * 4<br>
:<br>
:<br>
n! = 1 * 2 ** 3 * 4 * ... * n-1 * n</p>


<p>It's marked with an <strong>exclamation mark</strong>, and is equal to the <strong>product</strong> of all natural numbers from one up to its argument.</p>


<p>Let's write our code. We'll create a function and call it <codel>factorial_function</codel>. Here is the code:</p>

<codep class="syntax-color copy">def factorial_function(n):
    if n < 0:
        return None
    if n < 2:
        return 1
    
    product = 1
    for i in range(2, n + 1):
        product *= i
    return product


for n in range(1, 6):  # testing
    print(n, factorial_function(n))
</codep>

<br>

<p>Notice how we mirror step-by-step the mathematical definition, and how we use the <codel>for</codel> loop to <strong>find the product</strong>.</p>

<p>We add a simple testing code, and these are the results we get:</p>

<codep>1 1
2 2
3 6
4 24
5 120</codep>
