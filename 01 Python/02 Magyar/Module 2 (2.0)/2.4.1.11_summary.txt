<h1>A legfontosabb tudnivalók</h1>

<br>


<p>1. A <strong>változó</strong> egy adatok tárolására lefoglalt hely a memóriában, amire névvel hivatkozhatunk. A változó automatikusan létrejön (inicializálódik), amikor először rendelünk hozzá értéket. (2.1.4.1)</p>

<p>2. Minden változónak egyedi névvel, egy <strong>azonosítóval</strong>kell rendelkeznie. Az érvényes azonosító névnek aláhúzással (<codel>_</codel>), vagy betűvel kell kezdődnie, nem lehet üres karaktersorozat és nem lehet Python kulcsszó. Az első karaktert alulvonások, betűk és számjegyek követhetik. A Pythonban az azonosítók esetében a nagy- és kisbetűket megkülönböztetjük. (2.1.4.1)</p>

<p>3. A Python <strong>dinamikusan tipusos nyelv</strong>, ami azt jelenti, hogy nem kell változókat létrehozni (<i>deklarálni</i>) benne. (2.1.4.3) A változókhoz értékeket rendelhetünk egy egyszerű hozzárendelési operátorral, amely az egyenlőség (<codel>=</codel>) jel, például: <codel>var <mark style="background-color:#e6f2ff;">=</mark> 1</codel>.</p>


<p>4. A változókhoz rendelt értékek módosítására <strong>összetett értékadó operátorokat</strong> (gyorsító operátorokat) is használhatunk, például: <codel>var <mark style="background-color:#e6f2ff;">+=</mark> 1</codel>, or <codel>var <mark style="background-color:#e6f2ff;">/=</mark> 5 * 2</codel>. (2.1.4.8)</p>

<p>5. A már létező változókhoz új értékeket rendelhetünk az értékadó operátorokkal vagy az összetett operátorok egyikével. (2.1.4.5)</p>

<codep class="syntax-color">var = 2
print(var)

var = 3
print(var)

var += 1
print(var)
</codep>
<br>

<p>6. Szövegeket és változókat kombinálhatsz a <codel>+</codel> operátorral, és a <codel>print()</codel> függvényt használhatod a karakterláncok és változók kiírására. (2.1.4.4)</p>
<codep class="syntax-color">var = "007"
print("Agent " + var)
</codep>

<script>
function solution (id) {
    var element = document.getElementById(id);
    
if (element.style.display === "none") {
        element.style.display = "block";
    } else {
        element.style.display = "none";
    }
}
</script>
