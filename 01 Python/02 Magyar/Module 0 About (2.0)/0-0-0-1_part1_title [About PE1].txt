<h1>Üdvözlünk a Python alapok 1 tanfolyamon</h1>

<p>Ez a kurzus <b>egy két tanfolyamból álló sorozat első része,</b> amely a <a href="https://pythoninstitute.org/certification/pcep-certification-entry-level/" target="_blank">PCEP - Certified Entry-Level Python Programmer</a> (Belépő Szintű Python Programozó Tanúsítvány) és a <a href="https://pythoninstitute.org/certification/pcap-certification-associate/" target="_blank">PCAP: Certified Associate in Python Programming</a> (Kezdő Szintű Python Programozó Tanúsítvány) minősítő vizsgákra készít fel.</p>

<p>A tanfolyam fő célja, hogy a teljes programozási írástudatlanságból olyan szintű programozási ismeretekig juttasson el, amelyek lehetővé teszik számodra a Python nyelvű programok <b>tervezését</b>, <b>írását</b>, <b>hibakeresését</b>, és <b>futtatását</b>, valamint a <b>szoftverfejlesztési technológia alapfogalmainak</b> megértését.</p>

<p>A tanfolyam felkészít a széles körben értelmezett <b>szoftverfejlesztéssel</b> kapcsolatos állásokra és karrierre, amely junior fejlesztőként nemcsak magát a kód létrehozását, hanem a számítógépes <b>rendszerek tervezését</b> és a <b>szoftvertesztelést</b> is magában foglalja.</p>
