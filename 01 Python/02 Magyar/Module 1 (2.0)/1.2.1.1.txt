<h1>Hogyan szerezzük be a Pythont és hogyan kezdjük el használni?</h1>

<p>A Python 3-at többféleképpen is beszerezheted, attól függően, hogy milyen operációs rendszert használsz.</p>

<p><strong>A Linux-felhasználók gépén már valószínűleg telepítve van a Python</strong>, ez legtöbbször így van, hiszen a Python infrastruktúráját a Linux  operációs rendszer számos összetevője intenzíven használja.</p>

<p>Egyes disztribúciók sajátos eszközeiket összekapcsolhatják a rendszerrel, és sok ilyen megoldás, például a csomagkezelők, gyakran Python nyelven íródnak. A Linux világában elérhető grafikus környezetek bizonyos részei is használhatnak Pythont.</p>

<p>Ha Linux felhasználó vagy, nyisd meg a terminált/konzolt, és írd be:</p>

<codep>python3</codep>

<br>

<p>A shell promptban nyomd meg az <i>Enter</i>-t és várj!</p>

<p>Ha valami ilyesmit látsz:</p>

<codep>Python 3.4.5 (default, Jan 12 2017, 02:28:40)
[GCC 4.2.1 Compatible Clang 3.7.1 (tags/RELEASE_371/final)] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>></codep>

<br>

<p>akkor semmi mást nem kell tenned.</p>


<br>

</div>




<div class="small-12 large-6 columns">


<br>


<p>Ha nincs Python 3 a gépen, akkor a Linux dokumentációjában keresd meg, hogyan kell a csomagkezelőt használni egy új csomag letöltéséhez és telepítéséhez - a szükséges csomag a <strong>python3</strong>, vagy ezzel kezdődő nevű.</p>

<p>A nem Linux-felhasználók letölthetik a Pythont a <a href="https://www.python.org/downloads/" target="_blank">https://www.python.org/downloads/</a> weboldalról.</p>
