<h1>Kezdjünk bele a Python programozásba!</h1>

<p>Most, hogy telepítetted a Python 3-at, itt az ideje ellenőrizned, hogy működik-e, majd használatba venned.</p>

<p>Ez egy nagyon egyszerű eljárás, de pont elegendő ahhoz, hogy meggyőződj arról, a Python környezet telepítése sikeres, és a Pythonod működőképes.</p>

<p>A Python felhasználásának számos módja van, különösen, ha Python-fejlesztő leszel.</p>

<p>A munka megkezdéséhez a következő eszközökre van szükséged:</p>

<ul>
<li>egy <strong>szerkesztőprogram</strong>, amely támogatni fog a kód megírásában (jó, ha rendelkezik néhány speciális funkcióval, amely az egyszerű eszközökben nincs meg); ez a dedikált szerkesztőprogram többet nyújt, mint az operációs rendszer által kínált alapprogram;</li>
<li>egy <strong>konzol</strong>, amelyen elindíthatod az újonnan írt kódodat, és bármikor le is állíthatod, ha az irányíthatatlanná válik;</li>
<li>egy <strong>debugger</strong> nevű eszköz, amely képes lépésről lépésre futtatni a kódot, és lehetővé teszi, hogy a végrehajtás minden egyes pillanatában megvizsgáld azt.</li>
</ul>


<p>A Python 3 szabványos telepítése számos praktikus komponensén kívül tartalmaz egy nagyon egyszerű, de rendkívül hasznos alkalmazást, az IDLE-t.</p>

<p>Az <strong>IDLE</strong> egy mozaikszó: Integrated Development and Learning Environment (Integrált Fejlesztési és Tanulási Környezet).</p>
