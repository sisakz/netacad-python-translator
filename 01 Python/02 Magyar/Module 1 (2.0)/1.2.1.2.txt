<h1>A Python letöltése és telepítése</h1>

<p>Mivel a böngésző közli az oldallal, hogy milyen operációs rendszert használsz, egyetlen lépést kell csak tenned: kattints rá a kívánt Python-verzióra.</p>

<p>Ez esetben válaszd a Python 3-at. Az oldal mindig a legfrissebb verziót kínálja fel.</p>

<p>Ha <strong>Windows felhasználó</strong> vagy, indítsd el a letöltött .exe fájlt, és kövesd a lépéseket.</p>

<p>Egyelőre hagyd meg a telepítő által javasolt alapértelmezett beállításokat, egy kivétellel - keresd meg a <strong>Python 3.x hozzáadása a PATH-hoz</strong> nevű jelölőnégyzetet, és jelöld be.</p>

<p>Ez megkönnyíti a dolgokat.</p>

<p>Ha <strong>macOS felhasználó </strong>vagy, akkor a Python 2 verziója már lehet, hogy előre telepítve van a számítógépeden, de mivel mi a Python 3-mal fogunk dolgozni, ezért le kell töltened és telepítened kell a megfelelő .pkg fájlt a Python oldaláról.</p>
