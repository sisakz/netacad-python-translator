Gyakran ismételt kérdések (GYIK)

Kérjük, olvasd el ezt a GYIK-et és tekintsd át a tananyagot, mielőtt kapcsolatba lépnél az OpenEDG Python Institute munkatársaival. Köszönjük.
Utolsó frissítés: november 6, 2020
A tanfolyam verziója: 2.0


Ki támogatja a tanfolyamot?

A Python Institute hozta létre a PCAP: Programozási alapismeretek Pythonban (röviden: Python alapismeretek) tanfolyamot, a szakmai előmenetel javítása, fejlesztése és támogatása érdekében, a Python-programozás és a kapcsolódó hálózati technológiák terén.

A Python Institute ezt a tanfolyamot a Cisco Networking Academy programban részt vevő valamennyi intézmény számára ingyenesen kínálja oktató által vezetett és önállan elvégezhető képzésként. Ha többet szeretnél megtudni az OpenEDG Python Institute-ról, látogass el a www.pythoninstitute.org weboldalra.

Mik a kurzus tananyagának főbb jellemzői?

A tanfolyam felkészít a tágabb értelemben vett szoftverfejlesztéssel kapcsolatos állásokra és karrierekre, amely junior fejlesztőként nemcsak magát a kód létrehozását jelenti, hanem a számítógépes rendszerek tervezését és a szoftvertesztelést is.

A tanfolyam során hozzáférést kapsz gyakorlati tananyagokhoz, laborgyakorlatokhoz, kvízekhez, összefoglaló anyagokhoz és tesztekhez, hogy megtanuld, hogyan hasznosítsd a források tanulmányozása és a kódolási feladatok elvégzése során szerzett képességeket és ismereteket, és hogyan vegyél részt valós programozási kihívásokban és helyzetekben.

Tanfolyamunk elvégzése ugródeszkaként szolgálhat bármely más programozási nyelv elsajátításához, valamint a Python-t alapként használó technológiák (pl. Django) felfedezéséhez. Bevezető kurzusunkat a megfizethetőség, a barátságosság és a diákokkal szembeni nyitottság jellemzi. Az első rész az alapoktól indul, és lépésről lépésre vezet el a 2. részben elmagyarázott összetett problémákhoz, így olyan felelős szoftveralkotóvá válhatsz, aki az IT-iparág számos pozíciójában képes a különböző kihívásoknak megfelelni.

A kurzust elvégző hallgatók képesek lesznek alapszintű kódolási feladatok elvégzésére Python nyelven, és az objektumorientált programozásban használt alapvető fogalmak és technikák megértésére. Továbbá felkészülnek a Python Institute által kiadott PCEP - Certified Entry-Level Python Programmer (a Python alapismeretek 1 elvégzésével) és PCAP - Certified Associate in Python Programming (a Python alapismeretek 2 elvégzésével) tanúsítványok megszerzésére..

Miért tanuljak Pythont?

A Python mindenütt jelen van, az emberek számos Python-alapú eszközt használnak nap mint nap, akár észreveszik, akár nem. Több millió (nos, valójában milliárd) kódsort írtak már Pythonban, ami szinte korlátlan lehetőséget biztosít a kód újrahasznosítására és a jó példákból való tanulásra. Sőt, mi több, van egy nagy és nagyon aktív Python közösség, akik mindig szívesen segítenek.

Van még néhány tényező, ami miatt a Python kiválóan alkalmas a tanulásra:

  • Könnyen tanulható - a Python megtanulása rövidebb időt vesz igénybe, mint sok más nyelvé; ez azt jelenti, hogy gyorsabban el lehet kezdeni a tényleges programozást;
  • Könnyen használható új szoftverek írásához - Python használatával gyakran gyorsabban lehet kódot írni;
  • Könnyen beszerezhető, telepíthető és használható - a Python ingyenes, nyílt és multiplatformos; ezzel nem minden nyelv büszkélkedhet.

Ha még nem ismersz más programozási nyelveket, a Python rendkívül jó első nyelvnek, mert szilárd alapot ad, és lehetővé teszi, hogy más nyelveket (pl. C++, Java vagy C) sokkal könnyebben és gyorsabban tanulj meg. A Python tanulása szórakoztató és trendi!

Mennyi ideig tart a tanfolyam elvégzése?

TA tanfolyam körülbelül 70-80 óra alatt elvégezhető (35-40 óra kell a Python alapismeretek 1 és 35-40 óra kell a Python alapismeretek 2 elvégzéséhez). Ajánlott tanulási idő: 7-10 hét, heti 7-10 óra.

A tanfolyam illeszkedik valamelyik iparág által elismert tanúsítványhoz?

Igen, ez a tanfolyam megfelel a Python Institute PCEP - Certified Entry-Level Python Programmer (Python alapismeretek 1, 1-4. modul, PCEP-30-01 vizsga) és PCAP - Certified Associate in Python Programming (Python alapismeretek 2, 1-4. modul, PCAP-31-03 vizsga) tanúsítványainak. Mindkettő jó kiindulópont azoknak, akik professzionális (Python) fejlesztővé szeretnének válni.

Milyen értéket képviselnek az OpenEDG Python Institute tanúsítványai?

A Python-programozók iránti növekvő kereslet miatt fontos, hogy elismerést szerezz programozói szakértelmednek, és bizonyítani tudd szaktudásodat. Egy munkáltató nem csak a portfóliódat nézi meg, hanem különböző módokat keres a tudásod ellenőrzésére. A tanúsítvány gyakran az első szűrőeszköz, amelyet a munkaerő-felvételt végző személy használ, hogy segítsen kiválasztani azokat a jelölteket, akiknek lehetőségük lesz arra, hogy egy interjú során megmutassák képességeiket. Ha nem sikerül átjutni a legelső kiválasztási folyamaton, az még egy kiváló programozó számára is lehetetlenné teszi, hogy felfigyeljenek rá. Ez egy erős érv amellett, hogy szerezz tanúsítványt, és fordíts különös figyelmet az önfejlesztésre.

Az OpenEDG Python Institute tanúsítványa hatékony fegyver lehet a jobb foglalkoztatásért, a kiváló szakértelemért és a nagyobb versenyképességért folytatott versenyben.

A tanúsítvány egy jobb állás és egy jobb fizetés felé nyithat ajtót. Nagyszerű motiváció a tanulásra és az önfejlesztésre. Egyre több informatikai vezető által támasztott egyik legfontosabb követelmény, és gyakran a toborzók körében a jelöltekkel szemben támasztott standard kritérium.

Az OpenEDG Python Institute tanúsítványa igazolja a munkáltató számára, hogy rendelkezel az adott feladatok ellátásához szükséges szakértelemmel. Ugyanakkor azt is jelzi számukra, hogy hajlandó vagy bővíteni a tudásodat. És mivel a tanúsított személyek közvetlenül hozzájárulnak a vállalat hatékonyságának, termelékenységének és nyereségtermelő képességének növeléséhez, a Python Institute minősítés minden szervezet számára előnyös.

Lehet kedvezményt kapni a PCAP-31-03 minősítő vizsgára?

A Python Institute a Cisco Networking Academy program résztvevőinek, akik sikeresen elvégzik a PCAP: Programming Essentials in Python tanfolyam 1. és 2. részét, 50%-os kedvezményt biztosít a Pearson VUE vizsgaközpontokban és/vagy a Pearson VUE OnVUE Online Proctoring szolgáltatásában letett PCAP - Certified Associate in Python Programming (PCAP-31-03) minősítő vizsga listaárából.

A tanfolyamot sikeresen elvégző hallgatók, azaz akik:

  • sikerrel teljesítik a Python alapismeretek 1 - Összefoglaló tesztet (legalább 21/30 pont) és a Python alapismeretek 2 - Összefoglaló tesztet (legalább 21/30 pont),
  • sikeres Záróvizsgát tesznek (legalább 42/60 pont), és
  • kitöltik az elégedettségi felmérést,

jogosultak egy kedvezményes utalványra, amely 50%-kal csökkenti a vizsgadíjat. A kedvezményre jogosult hallgatók a főoldal alján, közvetlenül a FINAL TEST szakasz alatt tudják kitölteni az Utalványkérő űrlapot (Voucher Request Form).

A kedvezményes utalvány igényléséhez a tanulónak az Utalvány igénylési űrlap (Voucher Request Form) linkre kell kattintania, ki kell töltenie az űrlapot, el kell fogadnia az utalványokra vonatkozó szabályzatot, és el kell küldenie az igénylést. Azok a hallgatók, akik jogosultak a kedvezményre, az űrlap elküldése után azonnal megkapják a vizsgautalványt.

Szükségem van valamilyen eszközre a tanfolyamhoz?

A tanfolyam online elérhető bármilyen internetes böngészőn keresztül, Linux, Windows vagy Mac OS operációs rendszerű számítógépeken.

A legjobb tanulási élmény érdekében javasoljuk, hogy a Python 3 szabványos telepítése legyen a számítógépen. A Python 3 letölthető a https://www.python.org/downloads weboldalról. A telepítés tartalmaz egy IDLE (Integrated Development and Learning Environment) nevű szoftvert, amely lehetővé teszi egyszerű Python parancsok végrehajtását és a futó programok hatásának megtekintését.

A Linux-felhasználók gépén már valószínűleg telepítve van a Python, mivel a Python infrastruktúráját a Linux operációs rendszer számos összetevője intenzíven használja.

A Python beszerzéséről, telepítéséről és használatáról a PE 1 kurzus 1. moduljában teljes körű információ érhető el.

A tanfolyam tartalmával kapcsolatos kérdéseket ügyfélszolgálatunknak, a support@openedg.org e-mail címre küldheted el.
