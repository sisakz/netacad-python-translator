<h2>A Python alapok 1 kurzus tananyaga (PE1)</h1>



<p><strong>1. modul - Bevezetés a Python és a számítógép programozás világába</strong></p>
<ul>
<li>A Python, mint modern, univerzális és fejlődő programozási nyelv;</li>
<li>Python verziók és nyelvfejlesztés;</li>
<li>A Python programozás megkezdéséhez szükséges eszközök és környezetek rövid áttekintése.</li>

</ul>



<p><strong>2. modul - Adattípusok, változók, egyszerű input-output műveletek, alapműveletek</strong></p>

<ul>
<li>Hogyan írjuk meg és futtassuk a legelső Python programunkat;</li>
<li>Python literálok;</li>
<li>Python operátorok és kifejezések;</li>
<li>Változók elnevezése és használata;</li>
<li>Alapvető be- és kimeneti műveletek Python nyelven.</li>

</ul>


<p><strong>3. modul - Logikai értékek, elágazások, ciklusok, listák és listafeldolgozás, logikai és bitwise műveletek</strong></p>

<ul>
<li>Boolean adattípus;</li>
<li>Relációs operátorok Pythonban;</li>
<li>Döntéshozatal Pythonban: <i>if</i>, <i>if-else</i>, <i>if-elif-else</i>;</li>
<li>A kód ismétlődő végrehajtása ciklusok segítségével: <i>while</i> és <i>for</i>;</li>
<li>Logikai és bitwise műveletek Pythonban;</li>
<li>Listák Pythonban: felépítés, indexelés, szeletelés és a listaelemek kezelése;</li>
<li>Hogyan rendezzünk egy listát a <i>buborékrendezés</i> algoritmussal;</li>
<li>Többdimenziós listák és alkalmazásuk.</li>

</ul>



<p><strong>4. modul - Függvények, sokaságok, szótárak és adatfeldolgozás</strong></p>

<ul>
<li>A kód strukturálása és a függvények fogalma;</li>
<li>Függvényhívás és eredmény visszaadása egy függvényből;</li>
<li>Név hatókörök és változó elfedése;</li>
<li>Tuple - cél, felépítés és használat;</li>
<li>Szótár - cél, felépítés és használat.</li>

</ul>

<p><strong>Python alapok 1 - Összefoglaló teszt</strong></p>
<ul>
<li>1-4 modulok;</li>
<li>minimum 70%-os eredmény szükséges a Python alapok 2 megnyitásához</li>
</ul>

<h2>A Python alapok 2 kurzus tananyaga (PE2)</h1>

<p><strong>PE2 1. modul - Modulok, csomagok és PIP</strong></p>


<ul>
<li>Mi az a modul és miért van rá szükségünk?</li>
<li>Modulok importálása és használata;</li>
<li>Néhány hasznos Python modul áttekintése;</li>
<li>Mi az a csomag és miben különbözik a modultól?</li>
<li>Csomagok összeállítása és használata;</li>
<li>PIP - svájci bicska a csomagok karbantartásához.</li>

</ul>



<p><strong>PE2 2. modul - Kivételkezelés, karakterláncok, karakterlánc- és lista-metódusok</strong></p>

<ul>
<li>Karakterek, karakterláncok és kódolási szabványok;</li>
<li>Sorozatok vs. listák - hasonlóságok és különbségek;</li>
<li>Lista-metódusok;</li>
<li>Karakterlánc-metódusok;</li>
<li>Python-os módszer a futásidejű hibák kezelésére;</li>
<li>Hibakezelés felügyelete a <i>try</i> és <i>except</i> használatával;</li>
<li>A kivételek hierarchiája.</li>

</ul>



<p><strong>PE2 3. modul - Objektumorientált programozás Pythonban (OOP)</strong></p>

<ul>
<li>Az objektumorientált programozás alapfogalmai;</li>
<li>A strukturált programozástól az objektumorientált megközelítésig - motivációk és előnyök;</li>
<li>Osztályok, objektumok, tulajdonságok és metódusok;</li>
<li>Öröklődés és polimorfizmus;</li>
<li>Kivétel, mint objektum.</li>

</ul>



<p><strong>PE2 4. modul - Egyéb</strong></p>

<ul>
<li>Generátorok, iterátorok és cloure-ök;</li>
<li>Munka a fájlrendszeren, könyvtár struktúra és fájlok;</li>
<li>Python szabványos programkönyvtár modulok (os, datetime, time és calendar).</li>

</ul>

<p><strong>Python alapok 2 - Összefoglaló teszt</strong></p>
<ul>
<li>PE2 1-4 modulok;</li>
<li>minimum 70%-os eredmény szükséges a Záróvizsga letételéhez.</li>
</ul>
