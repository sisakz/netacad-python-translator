<h1>A legfontosabb tudnivalók: szótárak</h1>

<br>


<p>1. A szótárak rendezetlen<strong>*</strong>, változtatható és indexelt adatgyűjtemények. (<strong>*</strong>A Python 3.6x-ben a szótárak alapértelmezés szerint rendezetté váltak.)</p>

<p>Minden szótár <i>kulcs: érték</i> párok halmaza. A következő szintaxissal hozhatjuk létre:</p>

<codep class="syntax-color copy">my_dictionary = {
    key1: value1,
    key2: value2,
    key3: value3,
    }
</codep>
<br>

<p>2. Ha egy szótárelemhez akarunk hozzáférni, akkor ezt úgy tehetjük meg, hogy hivatkozunk a kulcsára egy szögletes zárójelben (példa 1), vagy a <codel>get()</codel> metódus, azaz tagfüggvény segítségével (példa 2):</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "kwiat": "flower",
    "woda": "water",
    "gleba": "soil"
    }

item_1 = pol_eng_dictionary["gleba"]    # ex. 1
print(item_1)    # outputs: soil

item_2 = pol_eng_dictionary.get("woda")    # ex. 2
print(item_2)    # outputs: water
</codep>
<br>

<p>3. Ha egy adott kulcshoz tartozó értéket szeretnénk megváltoztatni, akkor ezt az elem kulcsával történő hivatkozással tehetjük meg a következő módon:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

pol_eng_dictionary["zamek"] = "lock"
item = pol_eng_dictionary["zamek"]    
print(item)  # outputs: lock
</codep>
<br>

<p>4. Egy kulcs (és a hozzá tartozó érték) hozzáadásához vagy eltávolításához használjuk a következő szintaxist:</p>

<codep class="syntax-color copy">phonebook = {}    # an empty dictionary

phonebook["Adam"] = 3456783958    # create/add a key-value pair
print(phonebook)    # outputs: {'Adam': 3456783958}

del phonebook["Adam"]
print(phonebook)    # outputs: {}
</codep>
<br>

<p>Az <codel>update()</codel> tagfüggvénnyel is beilleszthetünk egy elemet a szótárba, és a <codel>popitem()</codel> tagfüggvénnyel eltávolíthatjuk az utolsó elemet, pl.:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {"kwiat": "flower"}

pol_eng_dictionary.update({"gleba": "soil"})
print(pol_eng_dictionary)    # outputs: {'kwiat': 'flower', 'gleba': 'soil'}

pol_eng_dictionary.popitem()
print(pol_eng_dictionary)    # outputs: {'kwiat': 'flower'}
</codep>
<br>

<p>5. A <codel>for</codel> ciklus segítségével végigmehetünk egy szótáron, például:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

for item in pol_eng_dictionary:
    print(item) 

# outputs: zamek
#          woda
#          gleba
</codep>

<br>

<p>6. Ha egy szótár kulcsain és értékein akarunk végighaladni, használhatjuk az <codel>items()</codel> metódust, például:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

for key, value in pol_eng_dictionary.items():
    print("Pol/Eng ->", key, ":", value)
</codep>
<br>


<p>7. Ha ellenőrizni szeretnénk, hogy egy adott kulcs létezik-e a szótárban, használhatjuk az <codel>in</codel> kulcsszót:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

if "zamek" in pol_eng_dictionary:
    print("Yes")
else:
    print("No")
</codep>
<br>

<p>8. A <codel>del</codel> kulcsszóval eltávolíthatunk egy adott elemet, vagy törölhetünk egy teljes szótárat. A szótár összes elemének eltávolításához a <codel>clear()</codel> tagfüggvényt kell használni:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

print(len(pol_eng_dictionary))    # outputs: 3
del pol_eng_dictionary["zamek"]    # remove an item
print(len(pol_eng_dictionary))    # outputs: 2

pol_eng_dictionary.clear()   # removes all the items
print(len(pol_eng_dictionary))    # outputs: 0

del pol_eng_dictionary    # removes the dictionary
</codep>
<br>

<p>9. Egy szótár másolásához használjuk a <codel>copy()</codel> metódust:</p>

<codep class="syntax-color copy">pol_eng_dictionary = {
    "zamek": "castle",
    "woda": "water",
    "gleba": "soil"
    }

copy_dictionary = pol_eng_dictionary.copy()
</codep>
