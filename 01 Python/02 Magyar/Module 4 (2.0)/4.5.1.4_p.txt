<h1>Egyszerű függvények: háromszögek és a Pitagorasz-tétel</h1>

<p>Nézd meg a kódot a szerkesztőben! Három értéket vár a felhasználótól, ezután a korábbi <codel>is_a_triangle</codel> függvényt használja. A kód készen áll a futtatásra.</p>

<p>A második lépésben megnézzük, hogy egy adott háromszög <strong>derékszögű-e</strong>.</p>

<p>Ehhez a <strong>Pitagorasz-tétel megfordítását</strong> kell felhasználnunk:</p>

<p><font size="+1"><strong>c<sup>2</sup> = a<sup>2</sup> + b<sup>2</sup></strong></font></p>

<p>Hogyan ismerjük fel, hogy a három oldal közül melyik az átfogó?</p>

<p><strong>Az átfogó a leghosszabb oldal</strong>.</p>

<p>Ime a kód:</p>

<codep class="syntax-color copy">def is_a_triangle(a, b, c):
    return a + b > c and b + c > a and c + a > b


def is_a_right_triangle(a, b, c):
    if not is_a_triangle(a, b, c):
        return False
    if c > a and c > b:
        return c ** 2 == a ** 2 + b ** 2
    if a > b and a > c:
        return a ** 2 == b ** 2 + c ** 2
    if b > a and b > c:
        return b ** 2 == a ** 2 + b ** 2
    return false



print(is_a_right_triangle(5, 3, 4))
print(is_a_right_triangle(1, 3, 4))
</codep>
