</codep>

<p>2. Egy függvényen belül létrehozott változó hatóköre a függvénytörzsre korlátozódik (4. példa).</p>

<p>4. példa:</p>

<codep class="copy">def adding(x):
    var = 7
    return x + var


print(adding(4))    # outputs: 11
<font style="color:red;">print(var)    # NameError</font>

</codep>
<br>


<p>3. A változó hatókörének globálissá tételéhez használjuk a <codel>global</codel> kulcsszót a változó neve előtt, pl: </p>

<codep class="syntax-color copy">var = 2
print(var)    # outputs: 2


def return_var():
    global var
    var = 5
    return var


print(return_var())    # outputs: 5
print(var)    # outputs: 5
</codep>

<p><strong>1. gyakorlat</strong></p>
<p>Mi történik, ha futtatod a következő kódot?</p>

<codep>def message():
    alt = 1
    print("Hello, World!")


print(alt)

</codep>

<br>

<div align="right">
<button class="button tiny" onclick="solution('sol')">Ellenőrzés</button>
</div>

<p id="sol" style="display: none;">
<codel>NameError</codel> kivételt vált ki (<codel>NameError: name 'alt' is not defined</codel>)
</p>

<script>
function solution (id) {
    var element = document.getElementById(id);
    
if (element.style.display === "none") {
        element.style.display = "block";
    } else {
        element.style.display = "none";
    }
}
</script>
