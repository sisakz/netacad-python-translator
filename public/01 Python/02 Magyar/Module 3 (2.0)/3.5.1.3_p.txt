<code>
my_list = []
swapped = True
num = int(input("How many elements do you want to sort: "))

for i in range(num):
    val = float(input("Enter a list element: "))
    my_list.append(val)

while swapped:
    swapped = False
    for i in range(len(my_list) - 1):
        if my_list[i] > my_list[i + 1]:
            swapped = True
            my_list[i], my_list[i + 1] = my_list[i + 1], my_list[i]

print("\nSorted:")
print(my_list)

</code>



<h1>Buborékrendezés – interaktív változat</h1>

<p>A szerkesztőben egy teljes programot látsz, amelyet a felhasználóval folytatott beszélgetés egészít ki, és amely lehetővé teszi a felhasználó számára, hogy megadja a lista elemeit és megkapja a rendezett számsort: <strong>Buborékrendezés – végleges interaktív változat</strong>.</p>

<p>A Python azonban rendelkezik saját rendezési mechanizmusokkal. Senkinek sem kell saját rendező algoritmust írnia, mivel elegendő számú <strong>kész eszköz</strong> áll rendelkezésre.</p>

<p>Ezt a rendezési módszert azért magyaráztuk el neked, mert fontos, hogy megtanuld, miként kell feldolgozni egy lista tartalmát, és hogy megmutassuk, miként működhet a valódi rendezés.</p>


<p>Ha azt akarod, hogy a Python rendezze a listádat, akkor ezt a következőképpen teheted meg:</p>

<codep>my_list = [8, 10, 6, 2, 4]
<mark style="background-color:#e6f2ff;">my_list.sort()</mark>
print(my_list)

</codep>
