EMPTY = "-"
ROOK = "ROOK"
board = []

for i in range(8):
    row = [EMPTY for i in range(8)]
    board.append(row)

board[0][0] = ROOK
board[0][7] = ROOK
board[7][0] = ROOK
board[7][7] = ROOK

print(board)

</code>

<h1>Listák a listákban: kétdimenziós tömbök – folytatás</h1>

<p>A tábla kiválasztott mezőjéhez való hozzáféréshez két indexre van szükség. Az első a sort választja ki, a második a soron belüli mező számát, amelyik de facto egy oszlopszám.</p>

<p>Vess egy pillantást a sakktáblára! Minden mező tartalmaz egy indexpárt, amelyet meg kell adni a mező tartalmának eléréséhez:</p>

<center><img src="/uploads/media/default/0001/01/ccde8f0849e18f228580e0367f6239866bd93998.png" width="75%"></center>

<br>
<br>

<p>A fenti ábrára pillantva helyezzünk el néhány sakkbábut a táblán. Először is tegyük fel az összes bástyát:</p>

<codep class="syntax-color">board[0][0] = ROOK
board[0][7] = ROOK
board[7][0] = ROOK
board[7][7] = ROOK
</codep>

<br>

<p>Ha egy huszárt akarunk tenni a C4-re, akkor a következőképpen járhatunk el:</p>

<codep class="syntax-color">board[4][2] = KNIGHT
</codep>

<br>

<p>És most egy gyalogot az E5-re:</p>

<codep class="syntax-color">board[3][4] = PAWN
</codep>

<br>

<p>Kísérletezz a kóddal a szerkesztőben!</p>
