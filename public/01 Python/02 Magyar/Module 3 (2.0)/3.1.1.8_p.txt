<h1>Mintakódok elemzése</h1>

<p>Most mutatunk néhány egyszerű, mégis teljes értékű programot. Ezeket nem fogjuk részletesen elmagyarázni, mert a kódon belüli megjegyzéseket (és a változóneveket) elegendő útmutatónak tekintjük.</p>

<p>Mindegyik program ugyanazt a problémát oldja meg: <strong>több szám közül megtalálja a legnagyobbat és kiírja</strong>.</p>

<p><strong>1. példa:</strong></p>

<p>Kezdjük a legegyszerűbb esettel: <strong>Hogyan azonosíthatjuk két szám közül a nagyobbat</strong>?</p>

<codep class="copy syntax-color"># Read two numbers
number1 = int(input("Enter the first number: "))
number2 = int(input("Enter the second number: "))

# Choose the larger number
if number1 > number2:
    larger_number = number1
else:
    larger_number = number2

# Print the result
print("The larger number is:", larger_number)
</codep>
 
<br>

<p>A fenti kódrészletnek egyértelműnek kell lennie a számodra: két egész értéket olvas be, összehasonlítja őket, és megállapítja, hogy melyik a nagyobb.</p>

<hr>

<p><strong>2. példa:</strong></p>

<p>Most pedig lássuk a Python egyik érdekes tulajdonságát, amelyet az alábbi kód szemléltet:</p>

<codep class="copy syntax-color"># Read two numbers
number1 = int(input("Enter the first number: "))
number2 = int(input("Enter the second number: "))

# Choose the larger number
if number1 > number2: larger_number = number1
else: larger_number = number2

# Print the result
print("The larger number is:", larger_number)
</codep>


<br>

<p>Megjegyzés: ha bármelyik <i>if-elif-else</i> ág csak egyetlen utasítást tartalmaz, akkor azt kódolhatjuk tömörebb formában is (nem kell a kulcsszó után behúzott sort készíteni, hanem csak folytassuk a sort a kettőspont után).</p>

<p>Ez a stílus azonban félrevezető lehet, és a jövőbeni programjainkban nem fogjuk használni, de mindenképpen érdemes ismerni, ha valaki más programjait akarjuk olvasni és érteni.</p>

<p>A kódban nincs más különbség.</p>

<hr>

<p><strong>3. példa:</strong></p>

<p>Itt az ideje, hogy egy kicsit bonyolítsuk a kódot. Keressük meg a három szám közül a legnagyobbat. Ez megnöveli a kód méretét? Egy kicsit.</p>

<p>Feltételezzük, hogy az első érték a legnagyobb. Ezután ezt a hipotézist a fennmaradó két értékkel ellenőrizzük.</p>

<p>Nézd meg az alábbi kódot:</p>

<codep class="copy syntax-color"># Read three numbers
number1 = int(input("Enter the first number: "))
number2 = int(input("Enter the second number: "))
number3 = int(input("Enter the third number: "))

# We temporarily assume that the first number
# is the largest one.
# We will verify this soon.
largest_number = number1

# We check if the second number is larger than the current largest_number
# and update the largest_number if needed.
if number2 > largest_number:
    largest_number = number2

# We check if the third number is larger than the current largest_number
# and update the largest_number if needed.
if number3 > largest_number:
    largest_number = number3

# Print the result
print("The largest number is:", largest_number)
</codep>

<br>

<p>Ez a módszer lényegesen egyszerűbb, mintha egyszerre próbálnánk megtalálni a legnagyobb számot, összehasonlítva az összes lehetséges számpárt (azaz az elsőt a másodikkal, a másodikat a harmadikkal, a harmadikat az elsővel). Próbáld meg újraépíteni a kódot magadnak!</p>
