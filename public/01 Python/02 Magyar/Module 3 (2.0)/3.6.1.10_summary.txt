<h1>A legfontosabb tudnivalók</h1>

<br>


<p>1. Ha van egy <codel>list_1</codel> nevű lista, akkor a <codel>list_2 = list_1</codel> értékadás nem másolatot készít a <codel>list_1</codel> listáról, hanem a <codel>list_1</codel> és <codel>list_2</codel> változók <strong>egy és ugyanazon listára mutatnak a memóriában</strong>. Például:</p>

<codep class="syntax-color copy">vehicles_one = ['car', 'bicycle', 'motor']
print(vehicles_one) # outputs: ['car', 'bicycle', 'motor']

vehicles_two = vehicles_one
del vehicles_one[0] # deletes 'car'
print(vehicles_two) # outputs: ['bicycle', 'motor']
</codep>

<br>

<p>2. Ha egy listát vagy annak egy részét szeretnéd másolni, azt <strong>szeleteléssel (slicing)</strong> teheted meg:</p>

<codep class="syntax-color copy">colors = ['red', 'green', 'orange']

copy_whole_colors = colors[:]  # copy the entire list
copy_part_colors = colors[0:2]  # copy part of the list
</codep>
<br>


<p>3. A szeleteléshez <strong>negatív indexeket</strong> is használhatsz. Például:</p>

<codep class="syntax-color copy">sample_list = ["A", "B", "C", "D", "E"]
new_list = sample_list[2:-1]
print(new_list)  # outputs: ['C', 'D']
</codep>
<br>

<p>4. A <codel>start</codel> és <codel>end</codel> paraméterek <strong>opcionálisak</strong> a <codel>list[start:end]</codel> szelet létrehozása során, például:</p>

<codep class="syntax-color copy">my_list = [1, 2, 3, 4, 5]
slice_one = my_list[2: ]
slice_two = my_list[ :2]
slice_three = my_list[-2: ]

print(slice_one)  # outputs: [3, 4, 5]
print(slice_two)  # outputs: [1, 2]
print(slice_three)  # outputs: [4, 5]
</codep>
<br>

<p>5. A <strong>szeleteket</strong> a <codel>del</codel> utasítással <strong>törölheted</strong>:</p>

<codep class="syntax-color copy">my_list = [1, 2, 3, 4, 5]
del my_list[0:2]
print(my_list)  # outputs: [3, 4, 5]

del my_list[:]
print(my_list)  # deletes the list content, outputs: []
</codep>
<br>

<p>6. Az <codel>in</codel> és <codel>not in</codel> kulcsszavak használatával tesztelheted, hogy egyes elemek <strong>léteznek-e a listában vagy sem</strong>, például:</p>

<codep class="syntax-color copy">my_list = ["A", "B", 1, 2]

print("A" in my_list)  # outputs: True
print("C" not in my_list)  # outputs: True
print(2 not in my_list)  # outputs: False
</codep>
