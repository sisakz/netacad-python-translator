<h1>Belépő szintű tanúsítvány: PCEP - Certified Entry-Level Python Programmer</h1>

<p>A <i>Python alapok 1</i> (PE1) a <i>PCEP - Certified Entry-Level Python Programmer</i> tanúsítványhoz igazodik.</p>

<p>A <a href="https://pythoninstitute.org/pcep-certification-entry-level" target="_blank">PCEP - Certified Entry-Level Python Programmer</a> minősítés egy olyan szakmai bizonyítvány, amely a vizsgázó képességét méri a <b>Python nyelvű programozás alapjaival </b>kapcsolatos kódolási feladatokban.</p>

<p>A vizsgázónak elegendő ismeretet kell mutatnia a számítógépes programozás <b>alapfogalmairól</b>, a Python nyelv <b>szintaxisáról és szemantikájáról</b>, valamint a Python <b>tipikus implementációs feladatok megoldásához szükséges készségekről</b>, a Python Szabványos Programkönyvtár segítségével.</p>

<p>A <i>PCEP - Certified Entry-Level Python Programmer</i> tanúsítvány azt mutatja, hogy a tanúsítvány tulajdonosa ismeri az olyan általános számítógépes programozási fogalmakat, mint az <b>alapvető operátorok, számrendszerek, Boole algebra, adattípusok, adatszerkezetek, függvények, elágazások, ciklusok</b>, valamint a Python programozási nyelv <b>szintaxisát, szemantikáját és futási környezetét</b>.</p>

<p>A <i>PCEP - Certified Entry-Level Python Programmer</i> tanúsítvány <b>egy köztes lépés</b> a <i>PCAP - Certified Associate in Python Programming</i> tanúsítvány felé, és <b>kiindulópont</b> a szoftverfejlesztői, Python programozói és a kapcsolódó technológiák területén való karrier elindításához.</p>

<p>A PCEP tanúsítvány megszerzése segít a tanúsítvány birtokosának kitűnni a többi jelölt közül, és bekerülni egy új munkahelyre.</p>
