<h1>Kezdő szintű tanúsítvány: PCAP - Python Certified Associate Programmer</h1>

<p><i>Python alapok 2</i> (PE2) a <i>PCAP - Python Certified Associate Programmer</i> minősítéshez igazodik.</p>


<p>A <a href="https://pythoninstitute.org/pcap-certification-associate/" target="_blank">PCAP - Python Certified Associate Programmer</a> tanúsítvány egy olyan szakmai minősítés, amely a <b>haladóbb szintű programozási feladatokon</b> méri a vizsgázó képességeit a Python programozásban, beleértve az <b>objektumorientált programozásban</b> használt alapfogalmak és technikák ismeretét.</p>

<p>A <i>PCAP - Certified Associate in Python Programming</i> tanúsítvány azt mutatja meg, hogy a tanúsítvány tulajdonosa ismeri a <i>PCEP - Certified Entry-Level Python Programmer</i> tanúsítvány által lefedett összes programozási fogalmat, úgy, mint a <b>modulok és csomagok, karakterláncok és karakterlánc metódusok, lista metódusok, kivételkezelés, objektumorientált programozás, generátor függvények, lambda függvény, fájlkezelés és a Python szabványos programkönyvtár (STL) kiválasztott moduljaival történő munka</b>.</p>

<p>A PCAP tanúsítvány megszerzése igazolja, hogy a jelölt teljes mértékben megismerte <b>a Python 3 által biztosított összes alapeszközt<b>, amelyek segítségével megkezdheti saját tanulmányait, és utat nyithat egy <b>fejlesztői karrier</b> felé.</p>
