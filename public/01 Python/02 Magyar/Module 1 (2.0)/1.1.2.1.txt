<h1>Mi az a Python?</h1>

<p>A Python egy széles körben használt, interpretált, objektumorientált, dinamikus szemantikájú, magas szintű programozási nyelv, amelyet általános célú programozásra használnak.</p>

<p>Annak ellenére, hogy a pitont talán óriáskígyóként ismered, a Python programozási nyelv neve a BBC egyik régi, a <strong>Monty Python Repülő Cirkusza</strong> című vígjátéksorozatából származik.</p>

<p>A siker csúcsán a Monty Python csapata világszerte élő közönség előtt adta elő jeleneteit, többek között a Hollywood Bowlban is.</p>

<p>Mivel a Monty Pythont a programozó két alapvető tápanyagforrása egyikeként tartják számon (a másik a pizza), a Python alkotója a tévéműsor tiszteletére nevezte el a nyelvet.</p>

<h1>Ki alkotta meg a Pythont?</h1>

<p>A Python egyik csodálatos tulajdonsága, hogy valójában egyetlen ember munkája. Általában az új programozási nyelveket sok szakembert foglalkoztató nagyvállalatok fejlesztik és teszik közzé, és a szerzői jogi szabályok miatt nagyon nehéz megnevezni a projektben részt vevő személyeket. A Python ez alól kivétel.</p>

<p>Nem sok olyan nyelv van, amelynek szerzői név szerint ismertek. A Pythont <strong><a href="https://en.wikipedia.org/wiki/Guido_van_Rossum" target="_blank">Guido van Rossum</a></strong> alkotta meg, aki 1956-ban született Haarlemben, Hollandiában. Természetesen Guido van Rossum nem maga dolgozta ki és fejlesztette a Python összes komponensét.</p>
