<h2>Szerezz PCEP tanúsítványt!</h2>

<p>A <i>Python alapok 1 tanfolyam</i> elvégzésével felkészülsz a <a href="https://pythoninstitute.org/pcep-certification-entry-level/" target="_blank">PCEP &ndash; Certified Entry-Level Python Programmer </a>(Kezdő Szintű Python Programozó Tanúsítvány) megszerzésére.</p>


<p>A <i>PCEP &ndash; Certified Entry-Level Python Programmer</i> minősítés azt igazolja, hogy ismered az olyan általános számítógép programozási fogalmakat, mint az <b>adattípusok</b>, <b>adatszerkezetek</b>, <b>függvények</b>, <b>elágazások </b> és <b>ciklusok</b>, valamint a Python programozási nyelv <b>szintaxisának</b>, <b>szemantikájának</b> és <b>futási környezetének</b> legfontosabb tulajdonságait.</p>


<img class="lightbox" src="/uploads/media/default/0001/02/23021d04887ddd6b79d34f0cab969d758b17295a.png" width="30%" alt="PCEP badge" style="float:left; padding:20px;">

A PCEP tanúsítvány megszerzése bizonyítja, hogy megismerkedtél a Python 3 által biztosított legalapvetőbb eszközökkel, amelyek lehetővé teszik, hogy középhaladó szinten kezd meg tanulmányaidat, és folytasd szakmai fejlődésedet.

<br><br>

A PCEP tanúsítvány lehetővé teszi, hogy igazold, nemcsak naprakész vagy a Pythonnal kapcsolatos kérdésekben, hanem hozzáértően tudod kezelni is azt.

<br><br>
Ráadásul az OpenEDG Python Institute révén hozzáférhetsz Python programozással foglalkozó szakemberek egy népes hálózatához, amely értékes erőforrás lehet a Pythonnal kapcsolatos problémákhoz és az innovatív megoldások kidolgozásához.



<br><br>

<div style="clear: both;"></div>

<br>

</div>

<div class="small-12 large-6 columns">

<br>

<p>A <i>PCEP - Certified Entry-Level Python Programmer</i> tanúsítvány <b>egy köztes lépés</b> a <i>PCAP - Certified Associate in Python Programming</i> tanúsítvány felé, és <b>kiindulópont</b> a szoftverfejlesztés, a Python-programozás és a kapcsolódó technológiák területén való karrier elindításához.</p>

<p>A PCEP tanúsítvány megszerzése segíthet a tanúsítvány birtokosának pl. egy állásinterjún kitűnni a többi jelölt közül.</p>
