<h1>A legfontosabb tudnivalók</h1>


<p>1. <strong>Literálnak</strong> hívjuk a kódban szereplő rögzített értékeket. A Pythonban többféle típusú literál létezik. Például egy literál lehet szám (numerikus literálok, pl. <codel>123</codel>) vagy karakterlánc (karakterlánc literálok, pl. "Én egy literál vagyok.").</p>

<p>2. A <strong>bináris rendszer</strong> egy olyan számrendszer, amelynek alapszáma a <i>2</i>. Ezért a bináris számok csak nullákból és egyesekből állnak, pl. az <codel>1010</codel>, ami a tízes számrendszerben <i>10</i>-et jelent. Az oktális és hexadecimális számrendszerek hasonlóan a <i>8</i>-as illetve a <i>16</i>-os alapszámot használják. A hexadecimális rendszer a decimális számokat és hat további betűt használ.</p>

<p>3. Az <strong>egész számok</strong> (vagy egyszerűen <strong>egészek</strong>) a Python által támogatott számtípusok egyike. Ezek olyan számok, amelyeket törtrész nélkül írunk, pl. <codel>256</codel> vagy <codel>-1</codel> (negatív egész számok).</p>

<p>4. A <strong>lebegőpontos szám</strong> a Python által támogatott másik számtípus. Ezek olyan számok, amelyek tartalmaznak (vagy képesek tartalmazni) törtkomponenst, például <codel>1.27</codel>.

<p>5. Aposztróf vagy idézőjel karakterláncon belüli kódolásához vagy az escape karaktert használhatod, pl. <codel>'I\'m happy.'</codel> vagy a karakterláncot a kódolni kívánt szimbólumokkal ellentétes szimbólumpárral nyithatod és zárhatod. Például <codel>"I'm happy."</codel> az aposztróf kódolásához, és <codel>'He said "Python", not "typhoon"'</codel> az idézőjel kódolásához.</p>

<p>6. A <strong>logikai értékek</strong> az igazságértékek ábrázolására használt két konstans objektum, a <codel>True</codel> és a <codel>False</codel>. A <codel>True</codel> numerikus értéke <codel>1</codel>, míg a <codel>False</codel> <codel>0</codel> értékű.</p>

<p><span class="label" style="background-color:orange;"><b>EXTRA INFORMÁCIÓ</b></span></p>

<p>A Pythonban még egy speciális literál használatos: a <codel>None</codel> literál. Ez a literál egy úgynevezett <codel>NoneType</codel> objektum, és <strong>egy érték hiányának</strong> jelzésére szolgál. Erről hamarosan többet fogunk mondani.</p>

<p><strong>1. gyakorlat</strong></p>
<p>Milyen típusú literálok vannak a következő két példában?</p>

<codep>"Hello ", "007"</codep>

<br>

<button class="button tiny" onclick="solution('sol')">Ellenőrzés</button>

<p id="sol" style="display: none;">
Mindkettő karakterlánc (karakterlánc literál).
</p>


<script>
function solution (id) {
    var element = document.getElementById(id);
    
if (element.style.display === "none") {
        element.style.display = "block";
    } else {
        element.style.display = "none";
    }
}
</script>
