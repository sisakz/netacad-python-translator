<h1>Some simple functions: evaluating the BMI</h1>

<p>Let's get started on a function to evaluate the Body Mass Index (BMI).</p>

<img src="/uploads/media/default/0001/01/c68be6d0a6bcec7d94c527db809dc7a9735ed0e4.png" width="75%" alt="BMI equals weight in kilograms divided by height in meters squared">

<br>
<br>

<p>As you can see, the formula gets two values:</p>

<ul>
<li>weight (originally in kilograms)</li>
<li>height (originally in meters)</li>
</ul>

<p>It seems that this new function will have <strong>two parameters</strong>. Its name will be <codel>bmi</codel>, but if you prefer any other name, use it instead.</p>

<p>Let's code the function:</p>

<codep>def bmi(<mark style="background-color:#e6f2ff;">weight</mark>, <mark style="background-color:#e6f2ff;">height</mark>):
    return weight / height ** 2


print(bmi(52.5, 1.65))

</codep>
