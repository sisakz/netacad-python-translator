<h1>Some simple functions: triangles and the Pythagorean theorem</h1>

<p>Look at the code in the editor. It asks the user for three values. Then it makes use of the <codel>is_a_triangle</codel> function. The code is ready to run.</p>

<p>In the second step, we'll try to ensure that a certain triangle is a <strong>right-angle triangle</strong>.</p>

<p>We will need to make use of the <strong>Pythagorean theorem</strong>:</p>

<p><font size="+1"><strong>c<sup>2</sup> = a<sup>2</sup> + b<sup>2</sup></strong></font></p>

<p>How do we recognize which of the three sides is the hypotenuse?</p>

<p><strong>The hypotenuse is the longest side</strong>.</p>

<p>Here is the code:</p>

<codep class="syntax-color copy">def is_a_triangle(a, b, c):
    return a + b > c and b + c > a and c + a > b


def is_a_right_triangle(a, b, c):
    if not is_a_triangle(a, b, c):
        return False
    if c > a and c > b:
        return c ** 2 == a ** 2 + b ** 2
    if a > b and a > c:
        return a ** 2 == b ** 2 + c ** 2


print(is_a_right_triangle(5, 3, 4))
print(is_a_right_triangle(1, 3, 4))
</codep>

<br>
