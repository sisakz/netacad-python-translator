<h1>Functions and scopes</h1>

<p>Let's start with a definition:</p>

<p>The <strong>scope of a name</strong> (e.g., a variable name) is the part of a code where the name is properly recognizable.</p>

<p>For example, the scope of a function's parameter is the function itself. The parameter is inaccessible outside the function.</p>

<br>

<p>Let's check it. Look at the code in the editor. What will happen when you run it?</p>

<p>The program will fail when run. The error message will read:</p>

<codep>NameError: name 'x' is not defined</codep>
<p style="text-align: right;"><font size="-2"><b>output</b></font></p>

<p>This is to be expected.</p>
