<code>
def strange_function(n):
    if(n % 2 == 0):
        return True

</code>



<h1>A few words about <font face="courier new">None</font>: continued</h1>


<p>Take a look at the code in the editor.</p>

<p>It's obvious that the <codel>strangeFunction</codel> function returns <codel>True</codel> when its argument is even.</p> 

<p>What does it return otherwise?<p>

<p>We can use the following code to check it:</p>

<codep class="syntax-color">print(strange_function(2))
print(strange_function(1))
</codep>

<br>

<p>This is what we see in the console:</p>

<codep>True
None</codep>
<p style="text-align: right;"><font size="-2"><b>output</b></font></p>

<br> 

<p>Don't be surprised next time you see <codel>None</codel> as a function result â€’ it may be the symptom of a subtle mistake inside the function.</p>
