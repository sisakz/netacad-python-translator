<h1>Key takeaways</h1>

<br>


<p>1. You can use the <codel>sort()</codel> method to sort elements of a list, e.g.:</p>

<codep class="syntax-color copy">lst = [5, 3, 1, 2, 4]
print(lst)

lst.sort()
print(lst)  # outputs: [1, 2, 3, 4, 5]
</codep>
<br>

<p>2. There is also a list method called <codel>reverse()</codel>, which you can use to reverse the list, e.g.:</p>

<codep class="syntax-color copy">lst = [5, 3, 1, 2, 4]
print(lst)

lst.reverse()
print(lst)  # outputs: [4, 2, 1, 3, 5]
</codep>

<p><strong>Exercise 1</strong></p>

<codep>lst = ["D", "F", "A", "Z"]
lst.sort()

print(lst)

</codep>

<br>

<script>
function solution (id) {
    var element = document.getElementById(id);
    
if (element.style.display === "none") {
        element.style.display = "block";
    } else {
        element.style.display = "none";
    }
}
</script>
