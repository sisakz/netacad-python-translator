<p>We said previously that the <codel>print()</codel> function separates its outputted arguments with spaces. This behavior can be changed, too.</p>

<p>The <strong>keyword argument</strong> that can do this is named <codel>sep</codel> (as in <i>separator</i>).</p>

<p>Look at the code in the editor, and run it.</p>

<p>The <codel>sep</codel> argument delivers the following results:</p>

<codep>My-name-is-Monty-Python.</codep>
<p style="text-align: right;"><font size="-2"><b>output</b></font></p>

<p>The <codel>print()</codel> function now uses a dash, instead of a space, to separate the outputted arguments.</p>

<p>Note: the <codel>sep</codel> argument's value may be an empty string, too. Try it for yourself.</p>
