<h1>The <span style="font-family:courier new;">print()</span> function â€’ keyword arguments</h1>


<p>And now it's time to try something more difficult.</p>

<p>If you look carefully, you'll see that we've used the <codel>end</codel> argument, but the string assigned to it is empty (it contains no characters at all).</p>

<p>What will happen now? Run the program in the editor to find out.</p>


<p>As the <codel>end</codel> argument has been set to nothing, the <codel>print()</codel> function outputs nothing too, once its positional arguments have been exhausted.</p>

<p>The console should now be showing the following text:</p>

<codep>My name is Monty Python.</codep>
<p style="text-align: right;"><font size="-2"><b>output</b></font></p>

<p>Note: <strong>no newlines have been sent to the output</strong>.</p>

<p>The string assigned to the <codel>end</codel> keyword argument can be of any length. Experiment with it if you want.</p>
