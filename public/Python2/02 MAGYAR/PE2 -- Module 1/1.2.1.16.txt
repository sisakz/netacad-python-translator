<h1>A Python Module Index</h1>

<p>Eddig csak a Python modulok alapjaival foglalkoztunk. A Python moduljai egy saját univerzumot alkotnak, amelyben maga a Python csak egy galaxis, és megkockáztatjuk azt állítani, hogy a modulok világának felfedezése lényegesen több időt vehet igénybe, mint az "alap" Python megismerése.</p>

<p>A Python-közösség világszerte több száz további modult hoz létre és tart fenn, amelyeket nagyon speciális alkalmazásokban használnak, például a genetikában, a pszichológiában vagy akár az asztrológiában.</p>

<p>Ezeket a modulokat nem terjesztik (és nem is fogják) a Pythonnal együtt, vagy hivatalos csatornákon keresztül, ami a Python univerzumot szélesebbé teszi - szinte végtelenné.</p>


<p>Az összes szabványos Python modulról itt olvashatsz: <a href="https://docs.python.org/3/py-modindex.html" target="_blank">https://docs.python.org/3/py-modindex.html</a>.</p>

<p>Ne aggódj - nem lesz szükséged az összes modulra, mivel sok közülük nagyon specifikus.</p>

<p>Mindössze annyit kell tenned, hogy megtaláld a kívánt modulokat és megtanuld, hogyan kell használni ezeket. Ez viszont egyszerű.</p>
