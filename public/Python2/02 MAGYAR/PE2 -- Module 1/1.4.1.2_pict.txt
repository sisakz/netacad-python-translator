    <h1>A Python csomagolási rendszere és annak használata (folytatás)</h1>

    <p>A korábban említett tároló (vagy röviden <i>repo</i>) neve <strong>PyPI</strong> (ez a Python Package Index rövidítése), és a Python Software Foundation részét képező Packaging Working Group nevű munkacsoport tartja fenn, amelynek fő feladata a Python fejlesztők támogatása a kód hatékony terjesztésében.</p>

<p>A weboldalukat itt találod:

<br>
<a href="https://wiki.python.org/psf/PackagingWG" target="_blank">https://wiki.python.org/psf/PackagingWG</a>.</p>

<br>

<p>A PyPI honlapja (amelyet a PWG kezel) a következő címen található:

<br><a href="https://pypi.org/" target="_blank">https://pypi.org/</a>.</p>

<br>

<p>Amikor 2020 júniusának elején egyszer benéztünk oda, azt találtuk, hogy a PyPI 237 515 projektnek adott otthont, amely 2 877 545 fájlból áll, és amelyet 427 487 felhasználó kezel.</p>

<p>Már ez a három szám is világosan mutatja a Python közösség erejét és a fejlesztői együttműködés fontosságát.</p>





<br>
<br>

<br><br>

    <p>Ki kell emelnünk, hogy a PyPI nem az egyetlen létező Python tároló. Épp ellenkezőleg, rengeteg van belőlük, amelyeket projektek számára hoztak létre és számos nagyobb és kisebb Python közösség felügyel. Valószínű, hogy egy nap Te és a kollégáid is szeretnétek majd <b>saját tárolókat létrehozni</b>.</p>

    <p>A PyPI egyébként a világ legfontosabb Python repója. Ha egy kicsit módosítjuk a klasszikus mondást, akkor kijelenthetjük, hogy "minden Python út a PyPl-hez vezet", és ez egyáltalán nem túlzás.</p>
