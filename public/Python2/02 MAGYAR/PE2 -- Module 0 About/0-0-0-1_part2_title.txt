<h1>Üdvözlünk a Python alapok 2 tanfolyamon</h1>

<p>Ez a kurzus <b>egy két tanfolyamból álló sorozat második része</b>, amely felkészít a Pearson VUE-nál letehető <a href="https://pythoninstitute.org/certification/pcap-certification-associate/" target="_blank">PCAP: Certified Associate in Python Programming</a> (Kezdő Szintű Python Programozó Tanúsítvány) minősítő vizsgára.</p>

<p>A tanfolyam ott folytatódik, ahol a <i>Python alapok 1</i> befejeződött. Fő célja, hogy megtanítsa a <b>Python programozás haladóbb aspektusaihoz</b> kapcsolódó készségeket, valamint az <b>általános kódolási technikákat</b> és az <b>objektumorientált programozást</b> (OOP).</p>

<p>A tanfolyamot azoknak a feltörekvő fejlesztőknek ajánljuk, akik a <b>szoftverfejlesztéshez</b>, <b>biztonsághoz</b>, <b>hálózatépítéshez</b> és a <b>tárgyak internetéhez</b> (IoT) kapcsolódó karrierlehetőségek iránt érdeklődnek.</p>
