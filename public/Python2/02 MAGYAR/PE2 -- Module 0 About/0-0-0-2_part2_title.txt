<h1>Miért érdemes PYTHON-t tanulnod?</h1>

<p>A Python mindenütt jelen van, az emberek számos Python-alapú eszközt használnak nap mint nap, akár észreveszik, akár nem. Több millió (nos, valójában milliárd) kódsort írtak már Pythonban, ami szinte korlátlan lehetőséget biztosít a kód újrahasznosítására és a jó példákból való tanulásra. Sőt, mi több, van egy nagy és nagyon aktív Python közösség, akik mindig szívesen segítenek.</p>

<p>Van még néhány tényező, ami miatt a Python kiválóan alkalmas a tanulásra:</p>

<ul>
<li>Könnyen tanulható - a Python megtanulása rövidebb időt vesz igénybe, mint sok más nyelvé; ez azt jelenti, hogy gyorsabban el lehet kezdeni a tényleges programozást;</li>
<li>Könnyen használható új szoftverek írásához - Python használatával gyakran gyorsabban lehet kódot írni;</li>
<li>Könnyen beszerezhető, telepíthető és használható - a Python ingyenes, nyílt és multiplatformos; ezekkel nem minden nyelv büszkélkedhet.</li>
</ul>

<p>A kurzus elvégzése ugródeszkaként szolgálhat bármely más programozási nyelv elsajátításához, valamint a Pythont alapként használó technológiák (pl. Django) felfedezéséhez. A tanfolyamot a megfizethetőség, a barátságosság és a diákokkal szembeni nyitottság jellemzi.</p>

<p>Az első rész az alapoktól indult, és lépésről lépésre vezetett el a bonyolultabb problémákhoz, amelyeket ebben a részben ismertetünk. Így egy olyan felelős szoftverfejlesztővé válhatsz, aki az IT-ipar számos pozíciójában képes a különböző kihívásoknak megfelelni.</p>
