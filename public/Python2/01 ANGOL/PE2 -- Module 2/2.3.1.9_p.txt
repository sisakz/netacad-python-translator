<h1>The <font face="Courier New">lower()</font> method</h1>

<p>The <codel>lower()</codel> method <strong>makes a copy of a source string, replaces all upper-case letters with their lower-case counterparts</strong>, and returns the string as the result. Again, the source string remains untouched.</p>

<p>If the string doesn't contain any upper-case characters, the method returns the original string.</p>

<p>Note: The <codel>lower()</codel> method doesn't take any parameters.</p>

<p>The example in the editor outputs:</p>

<codep>sigma=60</codep>
